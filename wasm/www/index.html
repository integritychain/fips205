FIPS 205 WASM Demo

This page is a simple demonstration of code from the FIPS 205 Rust crate running in the browser via wasm. This demo has a hardcoded random number generator so that its results can be compared to native test cases. Please see section 3.1 of the FIPS 205 standard for critical requirements on randomness when utilizing FIPS 205 in production.

Enter a message to sign below and click on 'Submit'. Keys will be generated, the message signed, and results displayed.
